# 🚀 Portfolio - Antony Salas

🌐 Demo en vivo: https://antony-s17.github.io/portfolio/

Este es mi portfolio personal como desarrollador web, donde muestro mis habilidades, proyectos y experiencia de forma interactiva y moderna.

---

## 🧭 Navegación del sitio

El sitio cuenta con un menú de navegación que permite acceder rápidamente a las distintas secciones del portfolio:

### 🏠 Inicio (Home)
Es la página principal del portfolio. Incluye:
- Presentación personal
- Resumen profesional
- Acceso rápido a proyectos y contacto
- Botones de acción (ver proyectos / contactarme)

---

### 👨‍💻 Sobre mí (About)
Sección donde detallo:
- Mi perfil como desarrollador Full Stack
- Experiencia profesional
- Educación
- Valores y enfoque de trabajo

---

### 🧠 Habilidades (Skills)
Muestra las tecnologías y herramientas que manejo, como:
- Frontend (HTML, CSS, JavaScript)
- Backend (PHP, Python, Node.js)
- Bases de datos y herramientas

Cada habilidad incluye nivel y descripción.

---

### 📁 Proyectos (Projects)
Sección donde muestro proyectos destacados con:
- Descripción del proyecto
- Tecnologías utilizadas
- Enlaces a repositorios o demos

---

### 📚 Recursos (Resources / Cheatsheets)
Área de aprendizaje y referencias donde organizo:
- Guías rápidas
- Cheatsheets de tecnologías
- Material de estudio

---

### 📬 Contacto (Contact)
Formulario para contactarme directamente:
- Nombre
- Email
- Mensaje

Este formulario está integrado con un servicio externo para envío de correos.

---

## 🌙 Funcionalidades del sitio

- 🌓 Modo oscuro / claro
- 📱 Diseño responsive (móvil, tablet, desktop)
- 🎨 UI moderna con efectos glassmorphism
- 📌 Navegación sticky
- ⚡ Transiciones suaves entre elementos

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3 (Flexbox + Grid + variables CSS)
- JavaScript (ES6+)
- GitHub Pages (deploy)
